CrowdCode • Multi-event GitHub Pages build

Turn a room full of phones into one shared experience.

This starter includes a host control room, audience phone experience, Firebase-powered live sync, reusable event presets, session launching, customizable reveals, mosaic mode, audio reactive settings, and GitHub Pages friendly static files.

Open Host Dashboard Open Audience Page
  • Create reusable events for multiple open houses, classroom reveals, admitted-student events, and conferences.
  • Launch a new live session with a fresh join code any time.
  • Use Firebase Realtime Database as the live state layer while keeping the site static for GitHub Pages.
Files included

index.html, host.html, audience.html, README.md

Setup

Paste your Firebase config into both HTML files, upload to a GitHub repo, and enable GitHub Pages.

Best first test

Create an event, launch a session, open audience.html on a phone, join, then start the countdown and reveal.
